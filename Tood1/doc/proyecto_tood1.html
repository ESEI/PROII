Proyecto Tood

Introducción

El proyecto Tood es un esqueleto de programa que se utilizará durante todo el curso para realizar ejercicios sobre él, practicando cuestiones que necesiten ser reforzadas.
El proyecto se distribuye como archivo *.zip, en el que se encuentran los archivos *.java necesarios para el proyecto en sí, así como las carpetas .idea y nbproject, que permiten abrir el proyecto en IntelliJ IDEA y Netbeans, respectivamente. En la carpeta doc/ se encuentra el enunciado del proyecto, mientras que en la subcarpeta doc/html/, reside la documentación (JavaDoc) del proyecto tal y como se entrega.

Objetivo del programa

El objetivo del programa consiste en guardar tareas a realizar, permitir modificarlas y marcarlas como terminadas. Por ejemplo, en un momento dado un listado de tareas podría ser el siguiente:
1. [ ] Limpiar la casa
2. [X] Hacer la compra
3. [ ] Estudiar PRO2

Diseño de la aplicación

La clase Bloc es la principal de la lógica de negocio, guardando una colección de tareas. La clase Tarea guarda toda la información sobre las tareas a realizar, mientras que Ilc se encarga de la interfaz del programa.

Ejercicios

  1. Modifica el método Ilc.menu(), de manera que visualice el número de entradas presentes en el Bloc, así como el número de entradas máximo.
  2. Crea el método toString() en la clase Bloc, de tal manera que recorra mediante un bucle todo el vector de tareas, y, llamando al método toString() de cada entrada, devuelva una cadena con toda la información. Modifica Ilc.leb(), de tal manera que utilice el nuevo método Bloc.toString().
  3. Añade el atributo prioridad a la clase Tarea. Los posibles valores serán BAJA, NORMAL, y ALTA.
  4. El método elimina() está vacío en este momento. Complétalo para que realmente funcione.
  5. En varios métodos (como en inserta()) se hacen comprobaciones de error, y en caso de que se produzca una situación excepcional se muestra un mensaje en la consola de errores, terminando a continuación el programa. Sustituye este comportamiento por un sistema de excepciones.
